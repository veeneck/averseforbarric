Data structure that hold severything we need to know about abilities specific to a squad and the unlock order.

  • unlocked

    Whether or not the ability has been unlocked.

    public var unlocked: Bool
  • abilityName

    Unique identifier of the ability. Squads can share abilities.

    public var abilityName: String
  • category

    Group abilities by category so that we can chain upgrades and require prereqs

    public var category: SRUnlockCategory
  • cost

    How many upgrade points this ability requires Specific to this squad.

    public var cost: Int
  • actionBarPosition

    Specific to this game. Used as metadata for UI to specify where the ability should show on the ability bar

    public var actionBarPosition: Int?
  • actionBarPriority

    Specific to this game. Used as metadata for UI to specify what order the overrides of UI are. For example, two abilities at same position. Priority 1 will show always unless it is locked and Priority 2 is valid Specifically, Attack would always show, but Charge would replace it if in Charge range and Charge is valid
