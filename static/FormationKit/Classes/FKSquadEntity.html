Agents
  • agentDidUpdate(_:)

    If in motion, always set the node position, and actual heading / position to that of the agent. Otherwise, ignore the agent. When at rest, the agent simulation will drift. By having actualPosition and actualRotation only update when moving we can ignore the drift when at rest.

    public func agentDidUpdate(_: GKAgent)
  • agentWillUpdate(_:)

    Currently there are no external factors like physics, so the agent does not need to be updated before each processing frame. However, it does still need to be set to actual location before logic executes, otherwise it will drift (the agent simulation drifts behind the scenes of GameplayKit)

    public func agentWillUpdate(_: GKAgent)
