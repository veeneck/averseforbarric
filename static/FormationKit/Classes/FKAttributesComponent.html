FKAttributesComponent

public class FKAttributesComponent : GKComponent

The goal of this component is to act as a gateway for all character attributes. Anything changed, read, or altered over time should be accessed from here.

Types of Attributes
  • Attribute

    Attributes that can be referenced and modified

    • Base: Empty placeholder.
    • WalkingSpeed: CharacterAgent.maxSpeed / Acceleration
    • HitPoints: Combat.Health
    • Strength: Combat.Strength
    public enum Attribute : String
