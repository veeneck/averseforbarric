Handles squad combat (engage, disengage, pick target, order units)

    Current target that this squad is attacking / enaging

    List of all squads attacking this squad. Only includes squads actively engaged (FKEngagedState) since this group is meant to dictate whether this squad is still locked in combat

    Basic way of tracking squad usage during combat. Each modifier applied thought CharacterEntity.CombatComponent will increment this by 1. Can make this more elaborate over time, but the main goal is to see whcih squads the player is using most.
