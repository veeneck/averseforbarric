Calculated Adjusted Locations
  • adjustLocations(_:rotation:translate:offset:random:reassign:)

    This will calculate the expected position for every unit, given the parameters. Updates the array of standing positions.

    internal func adjustLocations(_ position:CGPoint, rotation:Float, translate:float2? = nil, offset:CGPoint? = nil, random:Int? = nil, reassign:Bool = true) -> [float2]
    Parameters
    position

    The world position to start the calculations from
    rotation

    The heading that the formation should face, in radians
    translate

    If the rotation should translate to a specific unit, defaults to center
    offset

    After rotation, offsets to apply to the units
    random

    Add a small offset to each position to make it look more organic
    reassign

    Flag to permanently adjust the position. Leave false for previewing what move would look like
