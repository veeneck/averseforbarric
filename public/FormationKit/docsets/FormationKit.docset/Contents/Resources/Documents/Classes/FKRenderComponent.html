FKRenderComponent

public class FKRenderComponent : GKComponent

Controls the base visual display of an entity. This includes the sprite itself, along with updates to make sure the sprite is facing the proper direction and has the correct zPosition.

  • node

    The sprite node on the scene

    public var node: FKEntitySpriteNode?
  • shadow

    A shadow for the sprite

    public var shadow: FKPoolableObject?
  • pooledHealthBar

    A handle on the healthbar so that we don’t have to search the tree

    public var pooledHealthBar: FKPoolableObject?
  • basePosition

    Returns the standing position of the node where the feet touch the ground

    public var basePosition: CGPoint { get }
