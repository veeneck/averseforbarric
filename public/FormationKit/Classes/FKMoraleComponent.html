FKMoraleComponent

class FKMoraleComponent : GKComponent

Morale is a very specific attribute and has a ton of edge cases, so this component will handle that logic. Big concerns are:

  • Each unit has morale, but only the squad leaders morale affects fear / fleeing / etc
  • When leaders morale is changed, this component should be notified
  • Any state can trigger a flee. When engaged, the Flee action can be used. Likewise, a fear spell could be cast on a squad in reforming state
  • All logic to get out of the state and into the Flee state is here. Complete current command, disengage, etc
  • runFleeAbility()

    Flee is triggered two ways. One, when the ability is click by the user. In that case, just process the code. Two, dynamically (i..e: archers are engaged and roll to flee). When dynamic, call this function to treat Flee like a normal action.
