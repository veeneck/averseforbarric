Modifier

public struct Modifier

Whenever any AttributeComponent.Attribute needs to be modified it is done through a Modifier. This small struct can be attached to any object but is mainly used in Ability objects. Modifiers are attached to objects and the processed either instantly, after an attack finishes, over time, and so on. On its own, the Modifier struct does nothing.

  • ticks

    How many ticks before the modifier expires

    public var ticks: Int?
  • interval

    Time that should pass between each tick

    public var interval: Double?
  • duration

    If the modifier is not a DoT type effect, it can just have a fixed duration

    public var duration: Double?
  • timeElapsed

    Time when the current tick started

    public var timeElapsed: Double
  • name

    Name of effect to isolate when removing particle

    public var name: String
  • attribute

    Category of modifier which handles conditional splits

    public let attribute: FKAttributesComponent.Attribute
  • behavior

    Defines how the modifier will work throughout the code base

    public let behavior: Modifier.Behavior
  • modification

    Defines if the results of the modifier should permanently or temporarily change an attribute

    public let modification: Modifier.Modification
  • combatType

    Defines the type of modifier in terms of strnegth, weakness, resistance

    public var combatType: Modifier.CombatAttribute
  • combatSubtype

    Further breakdown of modifier type

    public var combatSubtype: Modifier.CombatAttributeSubtype
  • target

    What this modifier should apply to

    public var target: Modifier.Target
  • baseAmount

    The amount to modify by

    public var baseAmount: Float?
  • appliedBy

    A reference back to the Ability that applied this modifier. A modifier that comes with leveling up would not have any value here, but a modifier that came from an ability or an equipped item would.

    public weak var appliedBy: Ability?
  • origin

    A reference back to the position this modifier was spawned at. Useful to move objects around based on the position (i.e. catapult blow forces hit units backwards)

    public var origin: CGPoint?
  • force

    The force of the modifier. For example, a catapult blast would have highest, while a sword attack would be minor. Scale of 0 to 100

    public var force: CGFloat?
  • accuracy

    The chance that the modifier is successfully applied. 10 = 100% chance

    public var accuracy: Int

Type of Modifier
  • Behavior

    Hard coded options for how a modifier can behave. Used across the entire code base.

    • Instant: Instantly apply and forget
    • Duration: Instantly apply and then remove after duration
    • DoT: Apply with ticks overan interval
    • Permanent: Same as duration but never expires
    public enum Behavior : String
Type of Modification
  • Modification

    Hard coded options for how a modifier can behave. Used across the entire code base.

    • Temporary: Only alters attribute while modifier is active
    • Permanent: Alter the actual value of an attribute
    public enum Modification : String
Elemental or physical property of modifier
  • CombatAttribute

    Hard coded options for strengths and weaknesses of modifiers

    • Blunt: Strong aginst heavy armor
    • Piercing: Strong against light armor
    • Slashing: Strong against medium armor
    • Elemental: Not in use yet. Break down to ice, fire, etc?
      • Status: Status based modifier. Doesn’t factor in target armor, magic, etc. Just gets applied or resisted
    public enum CombatAttribute : Int
  • CombatAttributeSubtype

    Used to further classify an attack

    public enum CombatAttributeSubtype : Int
Target of Modifier
  • Target

    Most modifiers default to the Ability.Target. This is intended for abiltiies that affect a primary target, but then do secondary modifications. An example is charge. The target is .Myself, and +5 STR / +WalkingSpeed is added to the charging unit. But additionally -25 Morale should be added to the combat target

    • Ability: The target of the ability
    • CombatTarget: The target that the sqaud is engaging if there is one
    public enum Target : String
